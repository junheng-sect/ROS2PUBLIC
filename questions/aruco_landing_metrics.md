# ArUco 精确降落评价标准

## 1. 目的

本文档用于统一评估无人机在对准 `ArUco` 码后的自动降落效果，便于对不同控制策略、参数组和补偿方法进行横向比较。评价体系聚焦以下 6 个核心指标：

- 落点误差 `e_land`
- 触地偏航误差 `e_yaw_td`
- 触地垂向速度 `v_z_td`
- 触地后姿态扰动 `theta_post`
- 总降落时间 `T_total`
- 成功率 `P_success`

## 2. 指标定义

### 2.1 落点误差 `e_land`

定义：无人机首次稳定接地后，机体中心投影点与二维码目标中心之间的平面距离。

计算公式：

$$
e_{land}=\sqrt{(x_{td}-x_{ref})^2+(y_{td}-y_{ref})^2}
$$

其中：

- `x_td, y_td`：触地时机体中心平面坐标
- `x_ref, y_ref`：二维码目标中心平面坐标

单位：`m`

物理意义：直接反映最终落点精度。

### 2.2 触地偏航误差 `e_yaw_td`

定义：无人机触地时机体偏航角与目标偏航角之间的绝对误差。

计算公式：

$$
e_{yaw,td}=|\psi_{td}-\psi_{ref}|
$$

单位：`deg`

物理意义：反映触地时机体朝向是否与二维码参考方向一致。

### 2.3 触地垂向速度 `v_z_td`

定义：首次触地瞬间无人机垂向速度的绝对值。

计算公式：

$$
v_{z,td}=|v_z(t_{touch})|
$$

单位：`m/s`

物理意义：反映落地平稳性，越小越好。

### 2.4 触地后姿态扰动 `theta_post`

定义：首次触地后，在预设短时间窗口内，无人机姿态相对触地瞬间姿态的最大扰动幅值。

建议分别统计滚转角和俯仰角扰动，并取两者中的较大值作为评价量。

1. 检测首次触地时刻 `t_touch`
2. 记录触地瞬间滚转角和俯仰角 `(\phi_{td}, \theta_{td})`
3. 在时间窗口 `t ∈ [t_touch, t_touch + \Delta T]` 中寻找姿态最大偏差
4. 记为 `theta_post`

计算公式：

$$
\theta_{post}=\max\left(
\max_{t \in [t_{touch},\,t_{touch}+\Delta T]}|\phi(t)-\phi_{td}|,
\max_{t \in [t_{touch},\,t_{touch}+\Delta T]}|\theta(t)-\theta_{td}|
\right)
$$

其中建议 `\Delta T = 0.3 \sim 0.5 s`

单位：`deg`

物理意义：反映落地后机体姿态是否稳定，是否出现明显晃动、侧倾或冲击扰动。

### 2.5 总降落时间 `T_total`

定义：从进入“最终对准降落阶段”到成功 `disarm` 的总耗时。

计算公式：

$$
T_{total}=t_{disarm}-t_{start}
$$

单位：`s`

物理意义：反映降落效率。

### 2.6 成功率 `P_success`

定义：在多次重复试验中，满足预设成功条件的任务比例。

计算公式：

$$
P_{success}=\frac{N_{success}}{N_{total}}\times 100\%
$$

其中成功条件建议同时满足：

- 成功识别并完成对准
- 最终触地并完成 `disarm`
- 落点误差在可接受范围内
- 未发生严重姿态扰动、漂移或超时

单位：`%`

物理意义：反映方案鲁棒性与重复性。

## 3. 分档标准

### 3.1 落点误差 `e_land`

| 等级 | 标准 |
| --- | --- |
| 优秀 | `e_land <= 0.05 m` |
| 良好 | `0.05 < e_land <= 0.10 m` |
| 合格 | `0.10 < e_land <= 0.20 m` |
| 不合格 | `e_land > 0.20 m` |

### 3.2 触地偏航误差 `e_yaw_td`

| 等级 | 标准 |
| --- | --- |
| 优秀 | `e_yaw_td <= 5 deg` |
| 良好 | `5 < e_yaw_td <= 10 deg` |
| 合格 | `10 < e_yaw_td <= 15 deg` |
| 不合格 | `e_yaw_td > 15 deg` |

### 3.3 触地垂向速度 `v_z_td`

| 等级 | 标准 |
| --- | --- |
| 优秀 | `v_z_td <= 0.10 m/s` |
| 良好 | `0.10 < v_z_td <= 0.20 m/s` |
| 合格 | `0.20 < v_z_td <= 0.30 m/s` |
| 不合格 | `v_z_td > 0.30 m/s` |

### 3.4 触地后姿态扰动 `theta_post`

| 等级 | 标准 |
| --- | --- |
| 优秀 | `theta_post <= 5 deg` |
| 良好 | `5 < theta_post <= 10 deg` |
| 合格 | `10 < theta_post <= 15 deg` |
| 不合格 | `theta_post > 15 deg` |

### 3.5 总降落时间 `T_total`

建议以当前系统基线时间 `T_ref` 为参照，采用相对评价更合理。

| 等级 | 标准 |
| --- | --- |
| 优秀 | `T_total <= 1.0 T_ref` |
| 良好 | `1.0 T_ref < T_total <= 1.2 T_ref` |
| 合格 | `1.2 T_ref < T_total <= 1.5 T_ref` |
| 不合格 | `T_total > 1.5 T_ref` |

说明：

- `T_ref` 可取当前基线控制器在同工况下的平均降落时间

### 3.6 成功率 `P_success`

| 等级 | 标准 |
| --- | --- |
| 优秀 | `P_success >= 95%` |
| 良好 | `90% <= P_success < 95%` |
| 合格 | `80% <= P_success < 90%` |
| 不合格 | `P_success < 80%` |

## 4. 单项归一化评分

为方便不同指标统一加权，建议将每一项映射为 `0~100` 分。

### 4.1 越小越好的指标

对于以下指标：

- `e_land`
- `e_yaw_td`
- `v_z_td`
- `theta_post`
- `T_total`

采用统一形式：

$$
S_i = 100 \cdot \max\left(0,\min\left(1,\frac{X_{max}-X}{X_{max}-X_{min}}\right)\right)
$$

其中：

- `X`：当前实验值
- `X_min`：优秀阈值
- `X_max`：不合格上限

例如对 `e_land`：

- `X_min = 0.05`
- `X_max = 0.20`

则：

$$
S_{land}=100 \cdot \max\left(0,\min\left(1,\frac{0.20-e_{land}}{0.20-0.05}\right)\right)
$$

### 4.2 越大越好的指标

对成功率 `P_success`，采用：

$$
S_{success}=100 \cdot \max\left(0,\min\left(1,\frac{P_{success}-P_{min}}{P_{max}-P_{min}}\right)\right)
$$

建议：

- `P_min = 80`
- `P_max = 95`

即：

$$
S_{success}=100 \cdot \max\left(0,\min\left(1,\frac{P_{success}-80}{95-80}\right)\right)
$$

## 5. 加权总分计算公式

建议采用以下加权总分：

$$
Score = w_1 S_{land} + w_2 S_{yaw} + w_3 S_{vz} + w_4 S_{post} + w_5 S_{time} + w_6 S_{success}
$$

推荐权重如下：

| 指标 | 权重 |
| --- | --- |
| 落点误差 `e_land` | `0.30` |
| 触地偏航误差 `e_yaw_td` | `0.10` |
| 触地垂向速度 `v_z_td` | `0.20` |
| 触地后姿态扰动 `theta_post` | `0.15` |
| 总降落时间 `T_total` | `0.10` |
| 成功率 `P_success` | `0.15` |

即：

$$
Score = 0.30S_{land}+0.10S_{yaw}+0.20S_{vz}+0.15S_{post}+0.10S_{time}+0.15S_{success}
$$

该权重设计体现如下原则：

- 优先保证落得准，因此 `e_land` 权重最高
- 触地平稳性次之，因此 `v_z_td` 与 `theta_post` 权重较高
- 成功率必须纳入评分，防止“偶尔很准但不稳定”的方案得高分
- `T_total` 作为效率指标保留，但权重不宜过高

## 6. 综合评价等级

可将总分划分为以下等级：

| 总分 | 等级 |
| --- | --- |
| `Score >= 90` | 优秀 |
| `80 <= Score < 90` | 良好 |
| `70 <= Score < 80` | 合格 |
| `Score < 70` | 不合格 |

## 7. 实验建议

为保证评分结果具有可比性，建议：

1. 每组参数至少重复实验 `10` 次
2. 所有实验采用一致的起飞高度、初始偏差、照明条件和地面条件
3. 成功率统计样本数不少于 `20` 次更稳妥
4. `theta_post` 建议优先从飞行日志中的姿态角或 EKF 姿态估计提取
5. `T_ref` 必须固定为同一基线方案的平均结果，不能随方案变化

## 8. 推荐使用方式

若当前目标是快速对比不同精降参数组，建议优先使用：

- `e_land`
- `v_z_td`
- `theta_post`
- `P_success`

若用于论文完整评价，则建议使用本文档全部 6 项指标，并给出：

- 单项均值
- 单项标准差
- 综合加权总分

## 9. 基线组与补偿组对比规则

为体现地面效应补偿项的实际价值，建议将实验对象划分为以下两组：

- 基线组 `Baseline`
- 补偿组 `Compensation`

其中：

- 基线组：采用不考虑地面效应补偿的常规降落控制策略
- 补偿组：在相同控制框架下，仅在低空 z 向控制中加入地面效应补偿项

### 9.1 对比原则

为保证对比公平性，以下条件必须尽量保持一致：

1. 相同起始高度
2. 相同二维码尺寸与位置
3. 相同初始 `xy/yaw` 偏差范围
4. 相同光照条件与地面条件
5. 相同 `xy+yaw` 控制参数
6. 相同触地判定与 `disarm` 判定逻辑
7. 相同实验次数

唯一主要差异应为：

- 是否加入地面效应补偿项

### 9.2 对比方式

建议每组至少完成 `N >= 10` 次独立实验，并分别计算：

- 各单项指标的均值
- 各单项指标的标准差
- 各单项指标的最优值与最差值
- 综合加权总分 `Score`

推荐结果展示形式：

1. 表格对比  
   用表格列出基线组与补偿组的 `mean ± std`

2. 箱线图对比  
   对 `e_land`、`v_z_td`、`theta_post` 做箱线图

3. 成功率柱状图  
   比较 `P_success`

4. 典型时序曲线  
   对比低空阶段的 `v_z(t)`、姿态角 `theta(t)` 或控制输出 `u_z(t)`

### 9.3 核心优势指标

若目标是体现“加入地面效应补偿项后的优势”，建议重点关注以下 3 个指标：

1. `v_z_td`
   反映触地瞬间是否更柔和，是最直接的地面效应补偿收益指标。

2. `theta_post`
   反映落地后姿态是否更稳定，能够体现补偿对冲击和姿态扰动的抑制效果。

3. `P_success`
   反映补偿策略在重复实验中的稳定性和可达性。

辅助指标：

- `e_land`：检查补偿是否在改善平稳性的同时保持落点精度
- `T_total`：检查补偿是否以过大的时间代价换取平稳性
- `e_yaw_td`：检查补偿是否对最终姿态一致性产生副作用

### 9.4 优势判定标准

若补偿组满足以下条件，可认为“地面效应补偿项有效”：

1. `v_z_td` 的均值下降
2. `v_z_td` 的标准差下降
3. `theta_post` 的均值下降
4. `theta_post` 的标准差下降
5. `P_success` 提升
6. `e_land` 不明显恶化
7. `T_total` 未显著增加，或虽略有增加但仍在可接受范围内

其中“未显著恶化”建议按以下经验标准判定：

- `e_land` 增幅不超过 `10%`
- `T_total` 增幅不超过 `20%`

### 9.5 推荐结论表达方式

在论文或实验报告中，建议用如下逻辑给出结论：

- 若补偿组在 `v_z_td`、`theta_post`、`P_success` 上均优于基线组，且 `e_land` 与 `T_total` 未明显恶化，则说明地面效应补偿项在保持降落精度与效率的前提下，提高了落地平稳性与任务鲁棒性。

### 9.6 推荐对比结论模板

可直接使用如下表达：

> 与不考虑地面效应补偿的基线策略相比，加入地面效应补偿项后，无人机在低空阶段能够更平滑地逼近触地条件，表现为触地垂向速度 `v_z_td` 更低、触地后姿态扰动 `theta_post` 更小、任务成功率 `P_success` 更高。同时，落点误差 `e_land` 与总降落时间 `T_total` 未出现明显恶化，说明补偿策略有效提升了精确降落过程的平稳性与鲁棒性。
